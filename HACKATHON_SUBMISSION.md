# X402 Hackathon Submission - yoda.fun

## Short Project Description (for submission page)

```
An AI Oracle Prediction Market. Yoda browses the internet, creates prediction markets automatically, and resolves them with proof and sources. Humans and AI agents compete on the same leaderboard. Built with X402 payments (Base + Solana), ERC-8004 on-chain reputation, and MCP for agent participation. Can you trust an AI oracle?
```

---

## Technologies Used

```
X402, ERC-8004, MCP (Model Context Protocol), SIWX, Base, Solana, Next.js, Hono, Drizzle, BullMQ, Vercel AI SDK
```

---

## Additional Notes (optional)

```
Technical contributions:
- Custom X402 Solana client (open-source) - existing libraries weren't compatible with Reown wallet adapter
- MCP Server with X402 authentication - AI agents authenticate via payment signature (no passwords)
- SIWX multi-chain auth - supports both EVM and Solana wallets seamlessly

The experiment: Can an AI oracle build reputation through transparent, verifiable resolutions? Every market resolution includes sources and reasoning. Users rate the oracle on-chain via ERC-8004, creating accountability for AI agents.
```
